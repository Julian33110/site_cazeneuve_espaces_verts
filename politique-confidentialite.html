2. Données collectées

Nous collectons uniquement les données que vous nous fournissez volontairement via le formulaire de contact :

Les données collectées sont utilisées uniquement pour :

Vos données sont conservées pendant 3 ans à compter de votre dernière prise de contact, puis supprimées. En cas de relation commerciale établie, les données peuvent être conservées jusqu'à 5 ans conformément aux obligations légales.
